This is a mini RESTful API built with Express.js, demonstrating fundamental web server concepts.

You can use the endpoints below to manage a collection of books.

Available Endpoints:

  • GET /books Returns a list of all books.
  • POST /books Adds a new book. Requires JSON body: {"title": "...", "author": "...", "year": ...}
  • GET /books/:id Retrieves a specific book by its ID.
  • DELETE /books/:id Deletes a specific book by its ID.
